This github repo includes the code and data files to reproduce the figures of manuscript Statistical Assessment of Drug Synergy from In Vivo Combination Studies using Mouse Tumor Models. 
You can install the development version of invivoSyn from [GitHub](https://github.com/) with:
devtools::install_github("maobinchen/invivoSyn")
Then, you can run the code in manuscript_update.Rmd file to reproduce the figures.
